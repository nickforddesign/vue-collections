> A better collections plugin for Vue.js

## Installation

``` bash
npm install vue-collections
```

## Setup

```js
import Vue from 'vue'
import VueCollections from 'vue-collections'

Vue.use(VueColections)
```

## Build Setup

``` bash
# install dependencies
npm install

# serve demo at localhost:8080
npm start

# run tests with jest
npm test
```
